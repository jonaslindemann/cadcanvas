Description

Output device component for drawing a CadCanvas in a window

This unit contains the CadG32Device component for drawing the elements of a CadCanvas component in a window. The CadCanvas is assigned using the CadCanvas property of the component. This can be done directly in the design mode of the Delphi IDE.
