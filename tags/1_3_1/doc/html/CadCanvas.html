DescriptionusesClasses, Interfaces, Objects and RecordsFunctions and ProceduresTypesConstantsVariables

Description

Component for drawing CAD like elements on virtual canvas.

This unit contains the CadCanvas component for supporting a CAD like drawing surface. The unit also contains the classes implementing the supported CAD geomtry primitives.
